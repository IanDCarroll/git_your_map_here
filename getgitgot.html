What is Git?

Version Control System - it tracks the changes, preserving old versions & changes made. Makes for easier collaboration & effective combining of changes

What is GitHub?

Allows you to make changes to your version without affecting the shared project. Changes you make can be cleanly housed within your branch and then merged to the master branch when ready Logging into GitHub through the terminal
